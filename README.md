# AppList

## What AppList
android の アプリ一覧を表示するアプリです。
開発者がよく使う機能、設定画面に飛ぶ、通知をいじる、起動したことがある、ログインしたことがある など
アプリに対する操作のショートカット、状態の管理をお手伝いします。

## Why AppList
android の 設定アプリから特定のアプリの操作を行うことが手間であるため。
その手間を解決します。
